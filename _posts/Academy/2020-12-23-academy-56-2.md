## JSTL(JavaServer Pages Standard Tag Library)란?
>자바 서버 페이지 표준 태그 라이브러리는 Java EE 기반의 웹 애플리케이션 개발 플랫폼을 위한 `컴포넌트` 모음이다. JSTL은 JSP 페이지 내에서 자바 코드를 바로 사용하지 않고 로직을 내장하는 효율적인 방법을 제공한다.[^w1]
