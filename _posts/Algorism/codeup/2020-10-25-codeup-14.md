# [1253] a 부터 b까지 출력하기
> **[입력]** 3 8<br/>
  **[출력]** 3 4 5 6 7 8 

```java
import java.util.Scanner;
public class Main {
	public static void main(String[] args) {
		Scanner sc = new Scanner(System.in);
		int num1 = sc.nextInt();
		int num2 = sc.nextInt();
		if(num1 > num2) {
			int temp = num1;
			num1 = num2;
			num2 = temp;
		}
		for(int i = num1; i <= num2; i++) {
			System.out.print(i + " ");
		}
	}
}
```
